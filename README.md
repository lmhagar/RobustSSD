# RobustSSD
Supplementary code for An Efficient Framework for Robust Sample Size Determination manuscript 

There are three code files for this manuscript.

- 01_sec5p1: reproduces the results from Section 5.1 and Appendix C of the supplement
- 02_sec5p2: reproduces the results from Section 5.2 and Appendix D of the supplement
- 03_sec5p3: reproduces the results from Section 5.3 and Appendix E of the supplement
